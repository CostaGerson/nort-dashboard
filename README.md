# Nort Dashboard — Fase 0

Esqueleto do dashboard unificado da Nort Sports.

**Stack:** Next.js 14 (App Router) + PostgreSQL + Docker + NextAuth + Tailwind.

## O que tem nesta fase

- Login com usuário/senha (senhas com hash bcrypt no banco).
- Layout glassmorphism (claro + escuro) com sidebar e topbar.
- Página inicial com cards de placeholder (Calculadora, Mockup, Produção, Marketing).
- Banco PostgreSQL em container (apenas usuários por enquanto).
- Pronto para deploy via Docker Compose por trás do CloudPanel.
